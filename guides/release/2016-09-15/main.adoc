== Norsk [[norsk]]

I denne release er det fokusert på å konvertere implementasjonsguidene til et nytt format.
Formatet på guidene er endret fra PDF til HTML, og i den prosessen er også innhold samkjørt på tvers av guidene.
Dersom man skulle finne endringer som gjør at standardene endres er dette gjort ved uhell og vi ber om at man melder til inn til oss for retting.

== English [[english]]

This release focuses on converting our implementation guides to a new format.
Format is changed from PDF to HTML, and during the process are also content align between the guides.
Converting of guides is not intended to change the standards in any way, please notify if you find that this has happened by accident.
