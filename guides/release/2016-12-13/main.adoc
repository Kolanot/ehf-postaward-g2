== Norsk [[norsk]]

Denne utgivelsen inneholder utelukkende fjerning av regler som ikke har gitt noen verdi i produksjonssystemer. Basert på tilbakemelding har Difi vurdert det som formålstjenelig å slippe en utgivelse nå heller enn å vente til neste ordinære release.

Denne utgivelsen er bakoverkompatibel med utgivelsen i november, og det er valgfritt om man ønsker å oppgradere til denne.

Versjonsnummer er ikke oppdatert i denne utgivelsen.

== English [[english]]

This release includes solely the removal of rules that have not given any value in production systems. Based on feedback has Difi considered it appropriate to publish a new release now rather than waiting until the next scheduled release.

This release is backwards compatible towards the November release, and it is optional to upgrade to this release.

Version numbers are not updated in this release.
